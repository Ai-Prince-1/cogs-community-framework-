# Contributing to COGS

Thank you for helping improve the COGS Framework.

COGS is an open-source protocol for human flourishing.

The framework evolves through experimentation, observation, and community learning.

## Compatible Contributions

Examples include:

* improving language clarity
* adding examples
* improving facilitation techniques
* proposing new learning domains
* improving conflict resolution practices
* documenting successful experiments

## Breaking Changes

Changes that modify foundational safeguards require substantial review.

Examples include:

* removing the 7-person Circle limit
* creating permanent leadership positions
* introducing mandatory fees or tithing
* restricting fork rights
* centralizing governance
* creating proprietary ownership of the framework

## Core Principle

No proposal should increase dependency on a person, organization, platform, or authority.

COGS exists to strengthen human capability and community autonomy.
